# **App Name**: REDROOM: THE LAST EXIT

## Core Features:

- Kinematic Motion Physics Engine: Custom FPS framework utilizing Pointer Lock API for raw hardware mouse tracking and framerate-independent WASD movement physics.
- Organic Flesh-Shader Pipeline: Custom GLSL implementation of living, pulsating wall surfaces using 4-octave fractal noise and breathing sines.
- Procedural Soundscape Tool: A generative audio synthesis tool using the Web Audio API to modulate breathing drones and state-reactive heartbeats.
- Threadling Entity Intelligence: A sensory-driven AI tool featuring inverse-kinematics for 3D wall-crawling movement and an acoustic-based awareness system.
- Perceptual Geometry Matrix: A systemic layer that repositions unseen room objects when not within the camera's view vector to simulate spatial disorientation.
- Recursive Depth Orchestration: Infinite room loop logic that triggers environmental changes and matrix-collapse win states based on depth variables.
- Head-Bob & Gait Oscillations: Math-based kinematic camera sway using sinusoidal wave loops mapped to locomotion thresholds for immersive physicality.

## Style Guidelines:

- A dark, visceral scheme rooted in blood-and-sinew concepts. Primary color is a deep crimson (#b81414) for high intensity.
- Background is an oppressive, desaturated blackish-red (#1d1717) that creates high-contrast focus on the 'meat' textures.
- Accent color is a pale, sickly magenta-purple (#FF0000) providing an analogous but distinct contrast for interaction nodes.
- Headline font is 'Space Grotesk', bringing a machined, tech-focused precision. Pair with 'Inter' for UI labels and numeric tracking to maintain objectivity.
- Fullscreen minimalist immersion with a sparse HUD. Sensory indicators like 'acoustic awareness' are represented by chromatic aberration and screen noise rather than flat bars.
- Continuous, low-frequency UI pulsations mirror the rhythmic 'heartbeat' logic and living shaders of the environment.
- Brutalist, single-weight line icons only where absolutely necessary; most feedback is communicated via atmospheric distortion.
